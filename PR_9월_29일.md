## 되돌아보기
* `list.split(",")`: , 로 list = "a, b, c, d"를 나누어라
* json - 딕셔너리 형태랑 비슷(API-KEY)
* __a__: 함수랑 
* 은닉화 / 캡슐화

```py
a = "Apple, Banana, Cake, Donut"

def strip_and_upper(string_list):
  """
  이렇고 저렇고하는 함수 
  """
  result = []
  for string in string_list:
    result += [string.strip().upper()]
  
  return result

a = a.split(",")
print(strip_and_upper(a))
```
