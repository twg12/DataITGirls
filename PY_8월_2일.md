## 상태변경
* new_age = age + 1
    * age = age + 1
    * age += 1
* Other: age /= 2, age //= 2, age %= 2
* 파이썬 문법상 계속 상태변경이 가능하다

## 예시: Factorial
* Factorial *= 2, *= 3, etc
* 초기값: 1
* 누적: x부터 n까지의 합계
* ```py
    Factorial = 1 
    For x in range(1, n+1): 
        factorial *= x
    ```
* For * in *: for loop, 무언가의 기간동안
* 시작 <= x < 끝이기에 n까지 range에 포함 시키기 위해서 + 1을 붙여야한다. 
* ```py
    Factorial = 1 
    For x in range(1, n+1): 
        x = y
        factorial *= y
    ```

## Factorial Code
```py
f = 1

f *= 4

```
to
```py

print(f)
