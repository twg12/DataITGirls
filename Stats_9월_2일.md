# 선형모형
* MSE: numpy.mean((data - 예측) ** 2)
* from scipy.optimize import minimize
```py
def mse(m):
    return numpy.mean((data - m) ** 2)

minimize(mse, n)

# absolute value differences

def mae(m):
    return numpy.mean(numpy.abs(data - m))

minimize(mae, n)
```
